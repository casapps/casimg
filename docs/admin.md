# Admin Panel

The admin panel provides a web interface for managing casimg.

## Access

```
http://localhost:64580/admin
```

## First-Run Setup

On first access, you'll be prompted to create an admin account.

## Dashboard

The dashboard shows:
- Active conversions
- Queue status
- System resources
- Recent activity

## Configuration

Configure all server settings through the web interface:

- Server settings
- Conversion limits
- File retention
- Rate limiting
- Security options

## User Management

- Create user accounts
- Set permissions
- View activity logs

## Monitoring

- View conversion history
- Check system health
- Review error logs
- Monitor resource usage

## Maintenance

- Backup configuration
- Restore from backup
- Clear old files
- Update server

## Security

- Two-factor authentication
- Session management
- API key generation
- Access logs
